## Figma API Research (for Sunshine when she wakes)
**Key finding:** Figma REST API is READ-ONLY for design files. You can inspect, export, and get node data but NOT create or modify designs via REST API. Creating/modifying designs requires the **Plugin API** which runs inside Figma itself.

**Options for building carousels programmatically:**
1. **Figma Plugin** -- build a custom plugin that generates carousel frames from templates. Runs inside Figma. Most native.
2. **HTML/Canvas approach** -- generate carousel slides as HTML (using a template), screenshot them or export as images, then import to Figma. Simpler but less editable.
3. **Cursor + Figma workflow** -- design a carousel template in Figma, use Cursor/code to populate it with content via the Plugin API.
4. **Alternative: Skip Figma entirely** -- generate Instagram carousel images directly using HTML-to-image (puppeteer/playwright), which Janet can do natively without any new integration.

**Recommendation:** Option 4 is fastest -- I can generate carousel slides as images directly, no Figma API needed. But if she wants editability in Figma, Option 1 (custom plugin) is the way.
