## 1. BULK PRODUCT UPLOAD

### How It Works
- Desktop only (not mobile). Go to: Products > Add Products > Bulk Listing
- OR: Products > Manage Products > Bulk Action > Bulk Listing
- Templates are **category-specific** -- download a separate template per product category
- Max 1,000 products online at any time
- Max 5,000 SKUs can be corrected via the online bulk editor

### Downloading the Template
1. In Seller Center, go to the Bulk Listing page
2. Search for the product type or navigate the category tree
3. Select the leaf-level category (e.g. "Lip Color" not just "Beauty")
4. Optionally select brand (or "No Brand")
5. Click Download Template
6. Template downloads as `.xlsx` with category name + date in filename

**Important:** Since June 2025, templates are leaf-category-specific only. If listing products across different top-level categories (e.g. Beauty AND Bags AND Accessories), you need a separate template for each. Download max 3 at a time.

### Template Structure (Standard Columns)
The exact columns vary by category, but all templates include:

| Column | Required? | Notes |
|--------|-----------|-------|
| Product Name / Title | Yes | 25--200 characters |
| Category | Pre-filled | Don't change |
| Brand | Optional | "No Brand" if none |
| Description | Yes | Min 500 characters recommended |
| Main Image URL / Image 1--9 | Yes | At least 5 images for "Good" tier |
| Price | Yes | SGD |
| Stock Quantity | Yes | Per variant |
| SKU (Seller SKU) | Optional | Your internal reference code |
| Variation 1 Name | Conditional | e.g. "Shade", "Size", "Scent" |
| Variation 1 Values | Conditional | e.g. Strawberry Jam; Raspberry Coulis |
| Variation 2 Name | Conditional | For 2-tier variants (e.g. Size + Color) |
| Variation 2 Values | Conditional | |
| Price per Variant | Conditional | If variants have different prices |
| Stock per Variant | Conditional | |
| Weight (g) | Yes | For shipping calculation |
| Package Length (cm) | Yes | |
| Package Width (cm) | Yes | |
| Package Height (cm) | Yes | |
| Shipping Template | Yes | Pre-created template name |
| Product Status | Yes | Active / Inactive |
| Attributes | Varies | Category-specific (e.g. Shade, Finish, SPF) |

**Rules:**
- Do NOT add, delete, or rearrange rows/columns -- upload will fail
- Required fields are highlighted with a red border in the template
- Each variant goes on a new row
- After upload, fix errors via the online editor (correct up to 5,000 SKUs)

### File Requirements
- Format: Excel (.xlsx)
- No file size limit specified in documentation; keep it manageable
- Images must be hosted URLs (not local files) in the template -- or upload via the image column using Seller Center's image library

## 2. PRODUCT CATALOG REQUIREMENTS

### Title
- Length: 25--200 characters (hard limit); 50--80 characters recommended for SEO/display
- Must include: brand name (Analemma), product type, key identifying info
- NO: URLs, promotional language ("SALE!", "Best!"), email addresses
- Example: `Analemma Tinted Lip Ganache -- Strawberry Jam | Hydrating Tinted Balm`

### Description
- Minimum 500 characters recommended (longer = better listing quality tier)
- No seller contact info, website URLs, or other product cross-promotion
- Must include: key features, ingredients/materials, usage instructions, safety warnings if applicable
- For beauty: must be accurate, no exaggerated/false claims
- Organized in separate paragraphs; bullet points for readability

### Images
| Spec | Requirement |
|------|-------------|
| Minimum resolution | 600 x 600 px |
| Recommended resolution | 800 x 800 px or 1000 x 1000 px |
| Aspect ratio | 1:1 (square) |
| Format | JPG, JPEG, PNG |
| Max file size | 5 MB per image (some sources say 10 MB max per listing) |
| Min images | 1 (but 5+ required for "Good" tier listing quality) |
| Max images | 9 per listing |
| Main image | Must show full product, no watermarks, no promotional text overlay |
| Backgrounds | White or neutral recommended for main; lifestyle shots for additional |

**Listing Quality Tiers:**
- Poor: <3 images, low res
- Fair: 3--4 images
- Good: 5+ high-quality images -- gets better algorithmic placement

### Product Video (Optional but Recommended)
| Spec | Requirement |
|------|-------------|
| Resolution | 1080 x 1920 px (Full HD vertical) |
| Aspect ratio | 9:16 preferred |
| Format | MP4, MOV (H.264 codec) |
| Max file size | 500 MB (desktop upload) |
| Length | Up to 10 minutes; 15--60 seconds is optimal |

### Pricing
- Set in SGD
- Can set variant-specific prices
- Can add a "Compare at" / crossed-out price for promotions

### Inventory / Stock
- Set per variant
- Can set purchase limits per customer per variation
- Real-time inventory tracking in Seller Center dashboard

### SKU
- Optional internal reference
- Recommended: use consistent SKU codes (e.g. `ANA-LIP-SJ-001` for Strawberry Jam Lip Ganache)

### Attributes (Category-Specific)
These vary by category. For beauty products, expect:
- Shade/Color name
- Finish (matte, glossy, satin)
- Volume/Weight (net content)
- Key ingredients
- Skin type / suitable for
- Scent (for perfumes)

### Product Variants
- For products with variations (shades, sizes, scents): enable Product Variants
- Each variant gets its own price, stock, image, and SKU
- Variation types: up to 2 tiers (e.g. Shade + Size)
- Variation values: each on a new row in the bulk template

### Shipping Information (Required per Listing)
- Package weight (grams)
- Dimensions: L x W x H (cm)
- Assign a pre-created shipping template

## 4. SHIPPING SETUP

### Three Shipping Models Available in Singapore

**Option A: Ship by Seller (Self-Ship) -- Recommended for Analemma**
- You handle packing and drop-off/pickup
- Use your own preferred courier
- Must update tracking info manually in Seller Center
- Best control, most flexibility

**Option B: TikTok Shipping (Ship by TikTok)**
- Choose from TikTok's partnered logistics providers
- Tracking updates automatically
- Partners for Singapore include: **J&T Express** (official TikTok partner, integrated) and **Ninja Van** (SEA partner)
- You still pack and hand off to courier

**Option C: Fulfilled by TikTok (FBT)**
- TikTok warehouses and ships for you
- Handles storage, picking, packing, and delivery
- Best for high-volume sellers; may not be available for all product types initially

### Setting Up Shipping Templates
1. In Seller Center: Logistics > Shipping Templates (or Warehouse/Shipping Settings)
2. Create a template with:
   - Template name
   - Shipping method (standard, express)
   - Delivery regions
   - Shipping fee (free, flat rate, or weight-based)
3. Assign templates to products

### Recommended Couriers for Singapore Self-Ship
Based on e-commerce integration and pricing:

| Courier | Starting Price | Delivery | TikTok Integration |
|---------|---------------|----------|-------------------|
| J&T Express | SGD $3.50 | 1--3 days | Yes (official TikTok partner) |
| Ninja Van | SGD $3.49 | 1--3 days | Yes (SEA partner) |
| Pickupp | SGD $3.20 | 2--4 hours | Yes (TikTok integration) |
| EZIE | SGD $2.89 | 1--2 days | Yes (multi-platform) |
| SingPost | SGD $2.20+ | 2--3 days | Standard integration |

**Recommendation for Analemma:** Start with J&T Express or Ninja Van. Both integrate directly with TikTok Shop for auto-tracking updates and have low per-parcel rates suitable for small beauty/accessory items.

### Setting Up Warehouse Address
1. Logistics > Warehouses in Seller Center
2. Add pickup/return address
3. This is used for courier pickup scheduling

### Packaging Notes
- Beautifully branded packaging aligns with Analemma's aesthetic and increases unboxing content potential
- Include packing slips / order confirmations
- Ensure fragrance (perfume) packaging complies with courier restrictions for flammable liquids (alcohol-based perfume)

### Perfume/Fragrance Shipping Warning
Perfumes contain alcohol and are classified as **dangerous goods (DG)**. This affects shipping:
- Cannot ship by air in most cases
- Must use ground/road shipping couriers
- Declare as DG when booking shipments
- Check courier-specific DG policies before shipping perfumes
- Consider including Safety Data Sheet (SDS) documentation if requested

## 5. PAYMENT SETUP & PAYOUTS

### Linking Your Bank Account
1. In Seller Center: My Account > Account Settings > Bank Accounts
2. Click "Link your Settlement Account"
3. Verify identity
4. Select "Bank account" and enter details

**Requirements:**
- Account holder name must exactly match the shop owner/business name on registration
- Singapore local bank account (DBS, POSB, OCBC, UOB, Maybank, etc.)
- Corporate sellers: must use a business bank account
- For individual sellers: personal bank account is acceptable

**Alternative:** Wise or Payoneer multi-currency accounts work for receiving SGD if you prefer to separate business banking

### Fee Structure (Singapore, as of late 2025)

| Fee Type | Rate | Notes |
|----------|------|-------|
| Platform Commission | ~5.5% + GST = 5.995% | Varies by category; check Seller Center for exact rate |
| Transaction Fee | 3.27% | Effective from 1 Oct 2025 |
| Affiliate Commission | You set this | Optional; typically 10--30% of sale price |

**How to calculate your net payout:**
```
Commission base = Item price − Seller discount (your-funded discounts only)
Commission fee = Commission base × 5.995%
Transaction base = Buyer's actual payment (after all vouchers)
Transaction fee = Transaction base × 3.27%
Net payout ≈ Buyer payment − Commission fee − Transaction fee
```

**Example for Lip Ganache at SGD $30:**
- No discounts, no vouchers
- Commission: $30 × 5.995% = $1.80
- Transaction: $30 × 3.27% = $0.98
- Net payout ≈ $30 − $1.80 − $0.98 = **$27.22**

**Example for Perfume 100ml at SGD $168:**
- Net payout ≈ $168 − $10.07 − $5.50 = **$152.43**

### Payout Schedule
- TikTok settles after the order is confirmed received (return window closes)
- Standard settlement period: ~15--20 days after delivery confirmation
- Minimum withdrawal amount: varies; check Seller Center > Finance
- Payouts to linked bank account (SGD)

### GST Note
- TikTok charges 9% GST on platform fees (already included in the 5.995% commission rate shown above)
- You are responsible for your own GST obligations if your business is GST-registered

## 7. RETURN POLICY SETUP

TikTok Shop SG has a platform-level return/refund policy but sellers can configure their own:
- Go to: Seller Center > My Account > Return Settings (or Customer Service > After-sale Settings)
- Set: return window (typically 7--15 days), conditions, return address

**Standard policy to configure:**
- Returns accepted within 7 days of delivery for unused, unopened products
- Beauty products (lip, makeup): hygiene exception -- no returns on opened products
- Damaged/defective items: full refund or replacement
- Perfumes: no returns on opened/used bottles

**New in 2026:** TikTok now supports partial refunds, item exchanges, and replacements without returns -- configure these options in After-sale Settings.

## 9. BEAUTY PRODUCT COMPLIANCE (SINGAPORE)

### Singapore HSA Requirements
In Singapore, ALL cosmetic products must be **notified to the Health Sciences Authority (HSA)** before sale. This is a legal requirement under the Health Products Act and ASEAN Cosmetic Directive.

**Products that require HSA notification:**
- Tinted Lip Ganache (lip color/balm) -- YES, required
- Hair Oil -- YES, required
- Makeup Brush Set -- NO (not applied to skin)
- Perfumes -- YES, required

**How to notify:**
1. Register as a Responsible Person/Company on PRISM (HSA's online system)
2. Submit cosmetic product notification before selling
3. Product label must include: Product name, INCI ingredient list, net contents, manufacturer/distributor name and address, country of origin, "Made in [country]", warnings

**Labeling Requirements (ASEAN Cosmetic Directive):**
- Full INCI ingredient list (in descending order of concentration)
- Product name
- Net weight or volume
- Batch/lot number
- Expiry date or period after opening (PAO symbol)
- Country of origin
- Responsible person's name and address in Singapore
- Any required warnings

**For TikTok Shop specifically:**
- Category qualification may be required for Beauty & Personal Care -- check the Qualification Center in Seller Center
- Keep documentation ready: product labels, HSA notification confirmations, ingredient lists, supplier invoices

### Perfume-Specific Notes
- Perfumes with alcohol content are classified as dangerous goods for shipping
- On the listing, accurately disclose: alcohol content, fragrance notes, concentration (EDP/EDT/EDP)
- Concentration disclosure: 15ml travel size and 100ml full size -- list as Eau de Parfum if concentration is 15-20%+
